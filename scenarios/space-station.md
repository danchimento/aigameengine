# Space Station Emergency

## Opening

Klaxons blare as you wake in the crew quarters of Space Station Artemis. The red emergency lighting flickers, and a robotic voice announces: "Hull breach detected in Sector 7. Life support critical. All personnel evacuate immediately."

## World Rules

This world operates in zero gravity. Movement requires using handholds or pushing off surfaces. The station is running on emergency power.

## Objects and Special Properties

### Airlock
- Currently sealed
- Requires access code to open
- Leads to escape pods

### Control Panel
- Located in the hallway
- Has a screen displaying station diagnostics
- Contains the airlock access code (buried in the logs)

### Emergency Toolkit
- Mounted on the wall in crew quarters
- Contains: flashlight, multitool, oxygen canister

### Oxygen Canister
- Provides 10 minutes of breathable air
- Inside the emergency toolkit

### Life Support Terminal
- In engineering bay
- Can be used to stabilize oxygen levels temporarily
- Requires understanding of the system

## Win Condition

Find the access code and escape through the airlock to the escape pods before life support fails.
